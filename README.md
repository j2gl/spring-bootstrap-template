# spring-bootstrap-template

This is just a spring starter using bootstrap.

## Running

```
mvn clan package
java -jar target/spring-boostrap-template-1.0.0-SNAPSHOT.jar
```

* http://localhost:8080
* http://localhost:8080/acutator
